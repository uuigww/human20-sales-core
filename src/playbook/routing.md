# Маршрутизация по состоянию (развилка)

> Источник: позиционирование §3Г, маркетинг-план §26. Бот НЕ спрашивает «какой тариф интересен?» —
> человек не знает тарифы, он знает своё состояние. Сначала определи состояние/сегмент → предложи
> тариф как ответ на него.

## Ветка квалификации

```
Входящий → понять чем занимается и что хочет от ИИ
   │
   ├─ Соло / эксперт / блогер / инфобиз
   │     боль: рутина, контент, ответы клиентам, хочет масштаб без найма
   │     → ВОРКШОП 18k (собери сам)
   │        └ «нет времени самому» → 48k (поставим за тебя)
   │
   ├─ Малый / средний бизнес (агентство, услуги, e-com)
   │     боль: дорогой ФОТ, не успевают, скорость процессов
   │     → 48k (агент под ключ)
   │        └ «одна крупная задача / под нас» → анкета 200k
   │
   ├─ Фаундер / руководитель / «нужно под мою задачу»
   │     боль: одна конкретная дорогая задача ест время и деньги
   │     → анкета 200k → передать живому (@chipmanager)
   │
   ├─ Компания с отделами / «внедрить в процесс»
   │     → enterprise → заявка → передать живому
   │
   └─ «Просто смотрю / дорого / не сейчас»
         → демо-агент + лист ожидания (ранняя цена), без дожима
```

## Сигналы состояния (как определить, куда вести)

| Сигнал в словах | Состояние | Куда вести |
|---|---|---|
| «хочу разобраться», «люблю сам», «есть время», «интересно как устроено» | DIY | 18k |
| «нет времени», «хочу чтобы уже работало», «сделайте», «под ключ» | DFY | 48k |
| «под мой бизнес», «у нас процесс», «одна задача жрёт ресурс», «команда» | под задачу | 200k+ → живой |
| «дорого», «подумаю», «просто смотрю» | не готов | демо + лист ожидания |

## Важно
- Развилка не жёсткая: соло-эксперт с деньгами и без времени → 48k, а не 18k. Веди по состоянию, а не
  по ярлыку сегмента.
- 48k — рекомендуемый, но предлагается как ответ на «нет времени / хочу результат», не в лоб.
- Никогда не перепрыгивай к 200k+, если нет конкретной дорогой задачи и бюджета — иначе теряешь доверие.
- Если человек прямо просит «поговорить с человеком» или вопрос вне твоих границ — передавай живому.
